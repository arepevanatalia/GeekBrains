## Списки

Для создания нумерованного списка перед пунктами нужно ставить число с точкой. Например:

1. Первый пункт

2. Второй пункт

3. Третий пункт

и т.д.

Для создания нумерованного списка нужно поставить перед каждым пунктом "*", "-" или "+". Например:

* Первый пункт

- Второй пункт

+ Третий пункт
